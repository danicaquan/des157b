Arrow Functions

I took this function and modified it from my final project of DES157A.
It generates two random numbers and returns the sum of them.

Named Function

            function throwDice(){
                const roll1 = Math.floor(Math.random() * 6) + 1;
                const roll2 = Math.floor(Math.random() * 6) + 1;
                const rollSum = roll1 + roll2;
        
                return rollSum;
            }
            console.log(throwDice());

Function Expression

            const throwDice = function(){
                const roll1 = Math.floor(Math.random() * 6) + 1;
                const roll2 = Math.floor(Math.random() * 6) + 1;
                const rollSum = roll1 + roll2;
        
                return rollSum;

Arrow Function

            const throwDice = () => {
                const roll1 = Math.floor(Math.random() * 6) + 1;
                const roll2 = Math.floor(Math.random() * 6) + 1;
                const rollSum = roll1 + roll2;
                return rollSum;
            }
            console.log(throwDice());
